
# 中泰证券固定收益多资产分析平台

这是一个为金融分析师设计的简洁、高效的数据可视化平台。

## 功能特性
- **用户认证**：受保护的访问，仅限授权分析师。
- **债基久期跟踪**：从本地 `test.txt` 数据文件自动生成动态图表。
- **日期过滤**：支持通过日期范围筛选特定周期的久期走势。
- **研报展示**：精选微信公众号研报链接，支持原标题及发布日期展示。

## 默认测试账号
- **用户名**: `admin`
- **密码**: `1234`

## 后端实现思路 (建议使用 Python Flask)
由于您代码能力较弱，推荐以下极简实现：
1. **数据存储**：在项目根目录创建一个 `test.txt`。
2. **逻辑**：
   - 登录时，读取一个 `users.json` 文件或简单的字典映射来验证用户名。
   - 数据展示时，Flask 路由 `/api/data` 读取 `test.txt` 并以 JSON 格式返回给前端。
   - 文件上传：提供一个简单的上传页面，后端接收文件并覆盖 `test.txt`。

## 本地到云端部署指南 (免费)

### 第一步：本地准备
1. 确保已安装 Node.js。
2. 在项目目录运行 `npm install` 安装依赖。
3. 运行 `npm run build` 生成生产环境代码。

### 第二步：免费云端部署 (推荐 Vercel 或 Netlify)
这是目前最适合非专业开发者的方案：
1. **代码上传**：
   - 在 GitHub 上创建一个新仓库。
   - 将此代码推送到仓库。
2. **连接 Vercel**：
   - 访问 [Vercel.com](https://vercel.com) 并使用 GitHub 账号登录。
   - 点击 "New Project"，选择对应的 GitHub 仓库。
   - Vercel 会自动识别 React 环境并进行构建。
   - 点击 "Deploy" 即可获得一个 `xxx.vercel.app` 的免费域名。

### 第三步：建立自定义域名
1. **获取免费域名**：
   - 可以在 [Freenom](https://www.freenom.com/) 或类似的网站申请免费域名（如 .tk, .ml），或者在阿里云/腾讯云购买一个便宜的域名（如 .top, .site）。
2. **绑定域名**：
   - 在 Vercel 的项目设置 (Settings) -> Domains 中，输入你的域名。
   - 根据 Vercel 提供的指示，去你的域名服务商那里修改 DNS 记录（通常是添加一条 CNAME 记录指向 `cname.vercel-dns.com`）。
   - 等待几分钟，全球生效后即可通过你的域名访问。

## 维护建议
- **更新数据**：只需修改并重新上传 `test.txt` 到对应部署文件夹，或通过后端上传接口更新。
- **添加研报**：在 `constants.tsx` 的 `RESEARCH_REPORTS` 数组中添加新的对象即可。
